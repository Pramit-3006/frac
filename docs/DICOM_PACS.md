# PACS and DICOM Integration

## Supported Paths

- DICOM file upload
- DICOM metadata parsing
- DICOMweb WADO-RS retrieval
- C-STORE receiver design
- DICOM-SR export for structured reports

## Metadata

Tracked fields include PatientID, StudyInstanceUID, SeriesInstanceUID, SOPInstanceUID, modality, view position, body part examined, pixel spacing, and acquisition date/time.

## Safety

Research exports should pass through de-identification policies. Clinical workflows should preserve traceability through audit logs and access controls.
